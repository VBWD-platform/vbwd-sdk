## Problem

Today only the platform owner can put software into the GHRM catalogue: an admin creates a tarif plan (subscription plugin) and links a `GhrmSoftwarePackage` to it via `POST /api/v1/admin/ghrm/packages`. There is no vendor concept â€” `GhrmSoftwarePackage` has no owner, all revenue is the platform's. The goal is a **marketplace**: any user can become a software vendor. In the dashboard burger menu the vendor sees a top-level **"Software"** group with three sub-items:
- **My soft** â€” the vendor's published software listings (create, edit, submit for review),
- **My sales** â€” billing history of the vendor's packages,
- **My money** â€” current earnings balance + the payout methods that allow withdrawing it (PayPal etc., via S79).
